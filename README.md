# cn-sport-notes

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖体育相关笔记与资料整理。页面以静态 HTML 文件形式存放，不依赖特定域名或后端服务。

## 项目简介

cn-sport-notes 是一个轻量级的页面归档仓库，主要存放与体育领域相关的独立 HTML 页面。每个页面均为自包含的静态文件，可直接在浏览器中打开查看。仓库不针对任何具体网站或平台，仅作为个人笔记与资料的整理与备份。

## 目录说明

- `pages/` — 存放所有独立 HTML 页面文件，按主题或日期命名
- `assets/` — 页面引用的静态资源（如样式表、图片等），非必需时可为空
- `README.md` — 本说明文件

页面文件均以 `.html` 为后缀，文件名尽量体现内容主题，便于检索。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，内含样式与内容，无需外部依赖即可正常显示。
- 页面内容以体育相关笔记为主，包括但不限于赛事记录、规则整理、训练心得等。
- 归档遵循“一次提交，长期保留”原则，不主动删除已发布的页面（除非内容有误需修正）。
- 页面可能不定期更新或新增，历史版本可通过 Git 提交记录追溯。

## 维护说明

- 本仓库由个人维护，更新频率不固定。
- 欢迎通过 Issues 提交关于页面内容或格式的建议，但请注意本仓库不接收商业化推广或无关内容。
- 如需引用或转载页面内容，请注明出处并保持内容完整。
